# Eunoia-app
Yapay zekâ destekli, ders çalışma ve devam takibi için mobil uygulama. // Aplicación móvil para la gestión del estudio y asistencia con inteligencia artificial //

## Açıklama

Eunoia, zaman yönetimi araçlarını, görev organizasyonunu, çalışma materyallerini ve yapay zeka sistemini entegre eden bir uygulamadır.

## Özellikler

- Pomodoro Zamanlayıcı
- Görev ve Not Yönetimi
- Konu Organizasyonu
- PDF Belge Yükleme
- Yapay Zeka Sistemi
- İlerleme Takibi
- Konsantrasyon için İşitsel Destek Sistemi

## Teknolojiler

- Android Studio

## Descripción

Eunoia es una aplicación que integra herramientas de gestión del tiempo, organización de tareas, materiales de estudio y un sistema de inteligencia artificial.

## Funcionalidades

- Temporizador Pomodoro
- Gestión de tareas y notas
- Organización por asignaturas
- Carga de documentos PDF
- Sistema de inteligencia artificial
- Seguimiento del progreso
- Sistema de apoyo auditivo para la concentración

## Tecnologías

- Kotlin
- Android Studio
- Jetpack Compose
- API de inteligencia artificial
